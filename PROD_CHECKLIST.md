# 실제 배포(공개) 준비 체크리스트

소수 테스트 단계는 [필수 1~5]만, 불특정 다수 공개는 6~11까지.

## 필수 (안 하면 사고)
1. **유료 호스팅 + 디스크** — 무료는 데이터가 사라짐. Render Starter(월 ~$7).
2. **데이터 백업** — 디스크 고장/실수 대비(자료·계정 복구).
3. **OpenAI 비용 한도** — 사람 늘면 요금 폭증. 사용량 상한·1인당 질문 수 제한.
4. **업로드 제한** — PDF 크기·개수 제한(거대 파일로 비용/서버 폭주 방지).
5. **CORS 좁히기** — 지금은 모든 사이트 허용(`*`) → 내 도메인만 허용.

## 보안 / 계정 보완
6. **비밀번호 찾기 + 이메일 인증** — 지금은 비번 잊으면 끝, 가짜 이메일 가입 가능.
7. **레이트리밋** — 한 사람의 도배 요청 차단(봇·비용 방어).
8. **토큰 만료·로그아웃 점검** — 현재 토큰 유효기간 30일.

## 법 / 운영
9. **이용약관 · 개인정보 처리방침** — 학교 이메일·강의자료를 다루므로 필요.
10. **저작권 안내** — 강의자료 업로드 책임을 사용자에게 고지.
11. **에러 모니터링** — 서버 다운·오류 알림.

## 성능 (사용자 많아지면)
12. **검색 성능** — ChromaDB 단일 인스턴스 한계. 폭증 시 별도 벡터DB 검토.

---
현재 위치: 웹 배포(Render 무료)로 동작 확인 중 + 로컬 임베딩 OpenAI 전환 중.
다음: 1~5부터 셋업 → 필요 시 6~11.
